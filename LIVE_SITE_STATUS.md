## 🐛 Troubleshooting:

### ❌ Old Name Still Showing on Live Site

**Cause:** Browser cache or Vercel still deploying

**Solution:**
1. Wait 2-3 minutes for deployment
2. Hard refresh: Ctrl + Shift + R (Windows) or Cmd + Shift + R (Mac)
3. Clear browser cache
4. Try incognito/private window

### ❌ Articles Not Loading

**Cause:** Render backend sleeping

**Solution:**
1. Visit: https://jaldhara-rajsthan-ma2b.onrender.com/api/health
2. Wait 30-60 seconds to wake up backend
3. Refresh your site

### ❌ Admin Can't Login

**Cause:** Environment variables not set in Render

**Solution:**
1. Go to Render Dashboard
2. Check environment variables
3. Run seed script in Render Shell: `npm run seed`
